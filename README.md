
## 📊 Hotel Booking Cancellation Analysis

This project analyzes **hotel booking and cancellation trends** using historical booking data. An interactive **Excel dashboard** was created to visualize key metrics and understand patterns in customer booking behavior.

### 🔹 Project Highlights
- Developed an **interactive Excel dashboard** to analyze hotel booking and cancellation data.
- Performed **data cleaning and preparation** to ensure accurate analysis.
- Used **pivot tables and charts** to visualize important metrics such as **Total Bookings, Total Cancellations, Guest Types, and Room Preferences**.
- Analyzed **monthly booking trends** to identify peak booking periods and cancellation patterns.
- Compared **City Hotel vs Resort Hotel performance** to understand differences in booking behavior.
- Identified key factors influencing **hotel booking cancellations** to support better decision-making.

### 🎯 Key Metrics Analyzed
- **Total Bookings**
- **Total Cancellations**
- **Guest Type Distribution**
- **Room Type Preferences**
- **Monthly Booking Trends**
- **Hotel Type Comparison (City vs Resort)**

### 📌 Conclusion
The dashboard provides clear insights into **booking behavior, guest preferences, and cancellation trends**. These insights can help hotels improve **booking management, demand forecasting, and cancellation reduction strategies**.
